Quick Help Guide
System Navigation & Controls:
  • Search Shortcut

    Press CMD + K or focus the top bar search to quickly filter items.

  • Lending Rules

    Standard borrowing length is 14 days. Administrators can edit this on-the-fly during issuance.

  • Active Fine Rates

    Overdue items are assessed a fine of ₹5 per day, which accumulates dynamically until return.

  • Catalog Deletion

    Deletion can only be possible through database.
